# React Hooks 스터디 커리큘럼 ✍🏼

## 소개 ✨
이 저장소는 **React Hooks**를 차근차근 공부하고, 매주 작은 프로젝트로 실습해보는 과정을 기록합니다.  
기초부터 고급, 그리고 종합 프로젝트까지 총 8주 과정으로 진행합니다.


## 주차별 주제 🖐🏼

### Week 1: 기본 상태 관리
- **Hooks**: `useState`, `useReducer`
- **프로젝트**: Todo App  
- 간단한 상태 관리부터 reducer를 이용한 복잡한 상태 로직까지 연습

### Week 2: 상태 공유와 참조
- **Hooks**: `useContext`, `useRef`
- **프로젝트**: 테마 변경 메모 앱  
- 전역 상태 관리와 DOM 접근, 값 유지 패턴을 배움

### Week 3: 사이드 이펙트와 성능
- **Hooks**: `useEffect`, `useMemo`, `useCallback`
- **프로젝트**: 날씨 대시보드  
- API 호출, 값/함수 메모이제이션, 리렌더링 최적화 연습

### Week 4: 고급 이펙트와 레이아웃
- **Hooks**: `useLayoutEffect`, `useInsertionEffect`
- **프로젝트**: 드래그 앤 드롭 칸반 보드  
- DOM 측정과 스타일 삽입 타이밍 다뤄보기

### Week 5: 명령형 API와 디버깅
- **Hooks**: `useImperativeHandle`, `useDebugValue`, `useId`
- **프로젝트**: 커스텀 에디터  
- 부모에서 자식 메서드 제어, 디버깅, 접근성 향상

### Week 6: 최신 동시성 기능
- **Hooks**: `useTransition`, `useDeferredValue`
- **프로젝트**: 실시간 검색 앱  
- 부드러운 UX를 위한 논블로킹 업데이트와 값 지연 처리

### Week 7: 외부 시스템 동기화
- **Hooks**: `useSyncExternalStore`
- **프로젝트**: 실시간 채팅 앱  
- WebSocket/스토어와 React 상태를 동기화

### Week 8: 종합 프로젝트
- **Hooks**: 전체
- **프로젝트**: 개인 대시보드 앱  
- 지금까지 배운 모든 훅을 활용한 통합 프로젝트


## 진행 방식 🧑‍🤝‍🧑
**화요일**마다 공부 내용을 발표하고 공유합니다!
